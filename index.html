From sequence…

A protein begins as a string of letters.

Twenty amino acids, written in one dimension — the raw text of life.

…to structure

That string folds into a precise three-dimensional shape.

We predict, evaluate and compare structural models to understand how function arises from sequence.

Across genomes

Thousands of these molecules are encoded in microbial genomes.

Comparative genomics maps gene clusters, secretion systems, mobile elements and bacterial protein machinery.

Evolution rewrites it

Across billions of years, mutation and selection reshape these systems.

We trace where innovation happens — across protein families, genomic contexts, networks and deep time.

We build the computational tools to read these stories at genome scale.

Open code, curated data and reproducible, HPC-ready workflows.
