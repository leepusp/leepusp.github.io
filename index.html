From sequence…

A protein begins as a string of letters.

Twenty amino acids, written in one dimension — the raw text of life.

…to structure

That string folds into a precise three-dimensional shape.

We predict, evaluate and compare these folds to understand how form arises from sequence.

Across genomes

Thousands of these molecules are encoded in microbial genomes.

Comparative genomics maps secretion systems, mobile elements and bacterial protein machinery.

Evolution rewrites it

Across billions of years, mutation and selection reshape these systems.

We trace where innovation happens — across families, networks and deep time.

We build the computational tools to read these stories at genome scale.

Open code, datasets, tutorials and reproducible, HPC-ready workflows.
